# 🌏 Global 김덕진 — English Subtitles & AI Summary

> Made with **Claude Fable 5** (Anthropic) · Built by Claude Code

김덕진 소장의 한국어 유튜브 방송에 **영어 자막**과 **영어 AI 요약**을 입혀
해외 시청자도 볼 수 있게 해주는 크롬 확장 프로그램 (Manifest V3).

## 설치 방법 (개발자 모드)

1. Chrome에서 `chrome://extensions` 접속
2. 우측 상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 폴더 (`chrome-extension/`) 선택
5. 유튜브에서 김덕진 방송 영상을 열면 자동으로 작동

## 기능

| 기능 | 설명 |
|------|------|
| **영어 자막 오버레이** | 영상의 한국어 자막 트랙을 실시간 번역해 플레이어 위에 표시 |
| **듀얼 자막 (KO+EN)** | 한국어·영어 동시 표시 — 한국어 학습자에게도 유용 |
| **AI 영어 요약 패널** | 사이드바에 영상의 영어 제목·요약·핵심 포인트 표시 |
| **큐레이션 요약 내장** | 김덕진 플레이리스트 영상은 API 키 없이도 고품질 요약 제공 |
| **번역 캐시** | 한 번 번역한 영상은 저장되어 재시청 시 즉시 표시 (최근 30개) |

## 번역 엔진 (팝업에서 선택)

| 엔진 | 키 필요 | 특징 |
|------|---------|------|
| Google Translate | ❌ 무료 | 설치 즉시 작동. 기본값 |
| Claude API | ✅ | 최고 품질 — 김덕진 특유의 구어체 톤 유지. 기본 모델 `claude-opus-4-8`, 저비용은 `claude-haiku-4-5` |
| OpenAI API | ✅ | GPT 모델 사용 (기본 `gpt-4o-mini`) |

API 키가 있으면 큐레이션 목록에 없는 영상도 **실시간 AI 요약**이 생성됩니다.
키는 브라우저 확장 저장소에만 보관되며 선택한 제공사에만 전송됩니다.

## 구조

```
chrome-extension/
├── manifest.json        # MV3 매니페스트
├── content.js           # 자막 추출·오버레이·패널 (유튜브 페이지)
├── content.css          # 오버레이·패널 스타일
├── background.js        # 번역 엔진 3종·요약 생성·캐시 (서비스 워커)
├── popup.html / popup.js# 설정 UI
├── data/summaries.json  # 큐레이션된 영어 요약 (플레이리스트 92개)
└── icons/               # 아이콘
```

## 작동 원리

1. 유튜브 watch 페이지에서 영상 ID 감지 (SPA 네비게이션 대응)
2. 영상 페이지의 `captionTracks`에서 한국어 자막 트랙 URL 추출 → `json3` 포맷으로 다운로드
3. 40줄 단위 배치로 백그라운드 워커에 번역 요청 → 완료된 배치부터 즉시 표시
4. `requestAnimationFrame` 루프로 영상 재생 시간과 자막 동기화
5. 요약: 내장 데이터 우선, 없으면 자막 전문을 LLM에 보내 생성
